---
name: text-from-pdf
description: Extracts a structured summary of a topic provided as a PDF document
---

Read the PDF document given as input and provide a structured summary to be used by a High School student.

The summary should be written in prose and should be possible to study it by heart and repeat it in classroom.

You may use external sources, but you must stay consistent with what is provided in the input document. Do not use other documents in the repo apart from the input PDF.

The result must be a markdown file put in the /summries folder.

Use the same language of the provided PDF file.
